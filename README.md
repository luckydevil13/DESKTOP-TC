QUVNTNM TOOLCHAIN
===================================================
This toolchain is optimized for:
- x86_64 only
- Intel IVYBRIDGE CPU's (-mcpu=ivybridge -mtune=ivybridge)
- Linux Kernel 4.13.x

This toolchain uses:
- System zlib
- Linaro GCC 7.x (latest)
- Latest binutils
- Latest GLIBC 
- Latest components(isl/gmp/etc)
- Other components specified in ".config" file
